Ausführliche Beschreibung

A class for handling global timing issues.

This class provides support for central timing. In realtime mode it works as central timer and in batch mode it works as global time variable.

The time resolution is system dependent, but should not be worse than 1 ms in real time mode and 1μs in batch mode.
