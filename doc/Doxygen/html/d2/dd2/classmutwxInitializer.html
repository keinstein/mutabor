Ausführliche Beschreibung

An abstraction of wxInitializer.

the main purpose of this class is to increase the call stack for the objects that are created by the wxWidgets library.

When using ThreadSanitizer some suppressions are needed. Unfortunately it is not always possible to access the right function inside the binary blob of a shared object file. So we wrap them, here, and we add corresponding suppressions in the test folders.
