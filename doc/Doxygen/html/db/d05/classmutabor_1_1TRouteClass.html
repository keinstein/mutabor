Ausführliche Beschreibung

template<class I = InputDevice, class O = OutputDevice, class B = Box>
class mutabor::TRouteClass< I, O, B >

Class for managing routing.

As we are using smart pointers the Route gets deleted, when no pointers point to it any more. As we want to interactively manage routes, We must allow routes to have no input device attached to it.
