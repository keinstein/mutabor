Constructor for MutRouteWnd class.

This constructor sets up the Routing window. It loads Icons, initializes sizers and creates the shapes of the different routing elements (e.g., devices and boxes).

Parameter
parentparent Window as used in wxWidgets.
pospositions of the window (if unset use application default)
sizesize of the window (if unset use the application default).
