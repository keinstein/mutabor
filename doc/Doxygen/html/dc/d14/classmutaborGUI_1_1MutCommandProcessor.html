Ausführliche Beschreibung

Command processor for wxStyledTextControl.

This class proviedes a command forwarder for wxStyledTextControl. Since Scintalla uses its internal document/view framework it is overkill to implement a doubling. The best would be if wxSTC exposes the model to wxWidgets thus that it can be used easily.
